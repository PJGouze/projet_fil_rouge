MiniML – Application de prédiction nutritionnelle

MiniML est une application web permettant d'exploiter un modèle de Machine Learning pour estimer des indicateurs nutritionnels à partir de caractéristiques physico-chimiques des matières premières agricoles.

Fonctionnalités principales

  • Modèle XGBoost multi-sorties
  • Encodage automatique des variables catégorielles
  • Prédictions nutritionnelles multi-critères
  • Visualisation des performances du modèle
  • Historique des prédictions
  • Réentraînement dynamique

Navigation

Utilisez le menu à gauche pour :

  • Prédiction : saisir des données et générer une estimation nutritionnelle
  • Model Info : consulter les performances et paramètres du modèle
  • Historique : visualiser les prédictions précédentes
Ce projet est conçu à des fins pédagogiques et démonstratives dans un cadre Data Science & IA.
